## 객체 지향 프로그래밍 Object Oriented Programming (OOP)
데이터를 객체화해 부품처럼 쓸 수 있는 언어

#### 📌 객체화
데이터(속성) + 기능
→ 행동, 상태, 식별자을 부여

- 행동
- 상태
- 식별자


<b>객체화의 장점</b> <br>
- 유연성
- 자율성
- 유지보수성
- 가비지 컬렉터
- 다형성
- 캡슐화


## 동작 과정
- JDK Java Development Kit : 자바 개발 도구
- JRE Java Runtime Environment : 자바 코드가 실행되기 위한 최소 환경
- JVM Java Virtual Machine : .class 파일이 실행되기 위한 환경
